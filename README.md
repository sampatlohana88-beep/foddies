🍔 Foodies

A modern and responsive food ordering website built using HTML and CSS,  Foodies offers a clean and intuitive interface where users can browse delicious meals, search for their favorite dishes, and enjoy a smooth food ordering experience.

---

 ✨ Features

- 🍕 Responsive design for all screen sizes
- ❤️ Wishlist/Favorites
- 🎨 Modern and user-friendly interface
- ⚡ Fast loading and smooth animations
- 📱 Mobile-friendly design

 Technologies Used

- HTML5
- CSS3
